# Ruanm 项目

Ruanm 是一个专注于创造简单、美观、实用软件产品的项目品牌。我们致力于为用户提供卓越的数字体验，目前主要包含以下产品：

## 产品列表

### 1. Ruanm 屏保程序
- **类型**: Windows桌面屏保应用
- **功能**: 
  - 定时自动屏保
  - 动态壁纸切换
  - 时间日期自定义显示
  - 壁纸下载功能
- **技术栈**: Electron, HTML/CSS/JavaScript
- **了解更多**: [屏保程序详细介绍](program.html)

### 2. 阮铭泽工具箱
- **类型**: 多功能Windows桌面应用
- **功能**:
  - 天气预报查询
  - 日历功能
  - 科学计算器
  - 编程工具和资源
  - 系统知识查询
  - 实用网站推荐
- **技术栈**: Electron, HTML/CSS/JavaScript
- **了解更多**: [工具箱详细介绍](toolbox.html)

## 设计理念

我们始终坚持"用户体验至上"的设计理念，专注于：

1. **用户至上**: 始终将用户需求放在首位，深入了解用户痛点
2. **简约美学**: 追求简洁优雅的设计风格，去除冗余元素
3. **功能实用**: 注重产品的实用性，每个功能都为解决实际问题而设计

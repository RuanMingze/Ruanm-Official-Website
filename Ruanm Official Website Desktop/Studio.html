快速开始

简单几步即可在您的项目中使用 Ruanm Studio

使用 Ruanm Studio

// 1. 下载并解压 SDK 安装器
// 2. 运行安装器 exe 文件，自动安装创建窗口包、rpm 并添加到 Path

// 启动项目
rpm start

// RPM 包管理器使用示例
rpm install package-name
rpm uninstall package-name
rpm update package-name

// 项目文件结构
My_Ruanm_Studio_Project
├── bin
├── resources
│   └── index.html
├── config.json
└── README.md
